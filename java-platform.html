Java 的两大特性

Java是面对对象编程语言，具有以下显著的两大特性。

  1. 跨平台
    书写一次代码，可以在任何地方运行。
  2. 垃圾回收器
    有了垃圾回收器，Java会自动的将垃圾回收分配内存，程序员就不用太操心内存分配的问题。

JRE 和 JDK 的区别

JRE：java运行环境，包含了：

  • JVM
  • Java类库
  • 模块等

JDK：Java开发环境，是JRE的超集，有更多的工具，如：编译器，诊断工具等。

Java是解释执行么？

编译时：

源代码 -> javac编译 -> 字节码

运行时：

字节码 -> JVM -> 机器码

通常情况下， JVM 提供了 JIT (Just In Time) 动态编译器，JIT 可以在运行的时候将代码编译成机器码，属于编译执行，而不是解释执行！
