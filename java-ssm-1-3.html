Git可以干什么呢？

  1. 防止代码丢失
  2. 远程同步代码
  3. 团队合作利器
  4. 代码还原
  5. 记录代码版本

安装Git

Git的常用的操作命令

  • git init 初始化git仓库
  • git status 查看状态
  • git add 将文件添加到git仓库的暂存区
  • git commit 将添加到暂存区的文件提交到git仓库
  • git log 查看日志
  • git branch 查看当前的分支
  • git branch hi 创建一个叫做“hi”的分支
  • git checkout hi 切换分支到“hi”分支
  • git branch -d hi 删除“hi”分支
